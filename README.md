# Clay GTM Engineer

> An AI context pack that turns Claude Code into your personal Clay and GTM engineering expert.

## What This Is

Clone this repo, open it in Claude Code, and you have an expert assistant that knows Clay inside-out — from basic enrichment tables to production-grade outbound pipelines. Built on real GTM engineering frameworks and production-tested patterns, not generic documentation.

This isn't a chatbot or a wrapper. It's a knowledge base that makes your AI coding assistant deeply fluent in Clay and GTM workflow engineering. Ask it to design a scoring system, write a Clay formula, build a Claygent prompt, or architect an entire enrichment pipeline — and it responds with production-grade guidance grounded in real experience.

## Quick Start

1. **Clone the repo**
   ```bash
   git clone https://github.com/Mike-Mauricio/clay-gtm-engineer.git
   ```

2. **Open in Claude Code**
   ```bash
   cd clay-gtm-engineer
   claude
   ```

3. **Start building** — try any of these:
   - *"Help me build a lead enrichment table in Clay"*
   - *"Design a scoring system for my TAM"*
   - *"Write a Claygent prompt to find contact emails"*
   - *"Explain the difference between Tables and Audiences"*
   - *"Architect a waterfall enrichment pipeline with credit gating"*

## What's Inside

### Knowledge/
Reference documentation organized into two areas:
- **Clay/** — How Clay works: tables, enrichments, Claygent, audiences, integrations, credit optimization
- **GTM-Engineering/** — The thinking behind the builds: enrichment philosophy, scoring frameworks, segmentation design, outbound architecture

### Playbooks/
Step-by-step build guides, beginner to advanced:
1. **Your First Table** — Build a basic enrichment table
2. **Company Research** — TAM building and scoring
3. **Enrichment Pipeline** — Waterfall enrichment with credit gating
4. **Segmentation** — Signal convergence and routing logic
5. **Outbound Motion** — Personalization and message generation

### Examples/
Production-tested resources you can copy and adapt:
- **Formulas/** — Scoring, routing, cleaning, and gating formulas
- **Claygent-Prompts/** — Prompt templates for contact finding, company research, message drafting
- **Workflows/** — Complete end-to-end workflow specs
