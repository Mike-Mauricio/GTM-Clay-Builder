**Source:** Clay University course (6 lessons)

**Topic:** Using Clay directly inside AI chat tools for conversational prospecting — research, list building, enrichment, and outreach in one conversation

## Key Concepts Introduced

- **Conversational prospecting** — full motion in one AI chat window
- **Search → Filter → Enrich** — credit-saving 3-step pattern for MCP
- **Managed Functions** (Clay-built, auto-invoked) vs **Custom Functions** (ops-built, called by name)
- **Business Context** — workspace setting for company/ICP/value prop personalization
- **"Use Clay" prompt tip** — force Clay's live data providers vs generic web scraping
- **Generative email widget (Claude)** — send email directly from conversation
- **First-party data blending** — CRM + Gong + Audiences data in same conversation as market research
- **Individual contact briefs** — opinionated prioritization before outreach

## Who It's For

SDRs, AEs, and anyone doing prospecting/research. Separate Operators course exists for RevOps/admin setup.
