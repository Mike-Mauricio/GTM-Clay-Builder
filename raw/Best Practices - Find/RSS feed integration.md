Overview

## RSS Feed Overview

An RSS feed is a standardized web format that delivers automatic updates from websites, like blogs or news, to a user’s feed reader. The RSS Feed source allows you to pull the feed from a URLs into a Clay table in the form of rows.

## Action Import RSS feed into Clay as a source

**Step 1:** Add RSS Feed as a source

In a workbook, click `+ Add` at the bottom. Search for `RSS` and select from the results.

**Step 2:** Enter the Feed URL

This will import the feed’s items into individual rows.
